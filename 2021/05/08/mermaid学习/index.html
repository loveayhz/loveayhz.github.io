stateDiagram-v2
state 单据状态图 {
[*] --> 未提交: 员工 新增
未提交 --> 未提交:员工 修改
未提交 --> 待审批:员工 提交
待审批 --> 已审批:领导 审批通过
待审批 --> 已驳回:领导 审批驳回
已驳回 --> 已驳回:员工 修改
已驳回 --> 待审批:员工 提交
已审批 --> [*]
}

注意，这是状态图，不是流程图，状态图表达的是一个有限自动机,是状态之间的变迁
